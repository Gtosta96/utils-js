# **UTILS - JS** #

##### DependenciesTemplate #####
Utilizado para construção de tabela contendo tecnologias utilizadas na aplicação através o arquivo *package.json*.

##### Execução #####
Requer **Node.js**.
Para execução, execute o seguinte comando:
```sh
$ cd DependenciesTemplate
$ node script.run.js
```
